# YP-DS-08-Oil-prod
<h2>Описание проекта</h2>
Допустим, вы работаете в добывающей компании «ГлавРосГосНефть». Нужно решить, где бурить новую скважину. 
<br>Шаги для выбора локации обычно такие:
<ul>
<li>В избранном регионе собирают характеристики для скважин: качество нефти и объём её запасов;</li>
<li>Строят модель для предсказания объёма запасов в новых скважинах;</li>
<li>Выбирают скважины с самыми высокими оценками значений;</li>
<li>Определяют регион с максимальной суммарной прибылью отобранных скважин.</li>
</ul>
<br>Вам предоставлены пробы нефти в трёх регионах. Характеристики для каждой скважины в регионе уже известны. Постройте модель для определения региона, где добыча принесёт наибольшую прибыль. Проанализируйте возможную прибыль и риски техникой Bootstrap.
<h2>Описание данных</h2>
Данные геологоразведки трёх регионов находятся в файлах: 
<ul>
<li><b>/datasets/geo_data_0.csv</b></li>
<li><b>/datasets/geo_data_1.csv</b></li>
<li><b>/datasets/geo_data_2.csv</b></li>
<br>
<ul>
<li><b>id</b> — уникальный идентификатор скважины;</li>
<li><b>f0, f1, f2</b> — три признака точек (неважно, что они означают, но сами признаки значимы);</li>
<li><b>product</b> — объём запасов в скважине (тыс. баррелей).</li>
</ul>
